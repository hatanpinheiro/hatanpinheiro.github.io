EMENTA: Fundamentos de design de interação.Padrões de projeto Web. Especificação de documentos HTML. Folhas de estilos (CSS). Linguagem de back-end (PHP). OBJETIVO:
Desenvolver websites e sistemas web utilizando linguagens de front-end, HTML e CSS (lado cliente) quanto no backend PHP (lado servidor). METODOLOGIA: Serão utilizadas as seguintes estratégias e recursos nesta disciplina: Fórum, Webconference, Resolução de Exercícios e Desafios, material didático interativo, textos complementares e vídeos. CRITÉRIO DE AVALIAÇÃO: Instrumentos e critérios - Resolução - Percurso da trilha (20%) Critério: Participação e resolução (desafios e exercícios) em toda a trilha das unidades de aprendizagem. - Atividade avaliativa (20%) Critério: participação, fórum, resolução de exercícios. ATIVIDADES ACADÊMICAS: CONTEÚDO PROGRAMÁTICO Aula 1 - Criação e formatação de páginas web com HTML - Criação de páginas web com HTML5 Aula 2 - Criação e validação de formulários com HTML5 - Criação e formatação de páginas web com CSS Aula 3 - Criação de páginas com CSS3 - Definição do sistema Aula 4 - Projetando site - Comportamento com JavaScript Aula 5 - Ajax - Linguagem PHP Introdução Aula 6 - Linguagem PHP  Estruturas de controle - Linguagem PHP  Outros comandos Aula 7 - Linguagem PHP com formulários - Linguagem PHP, Mysql e funções de CRUD Aula 8 - Linguagem PHP com framework - Linguagem PHP Projeto
ATIVIDADES ACADÊMICAS Realização de toda a trilha de aprendizagem do material na plataforma, com participação nas atividades e resolução dos desafios e exercícios. Realização das atividades presenciais e prova presencial conforme calendário acadêmico.
